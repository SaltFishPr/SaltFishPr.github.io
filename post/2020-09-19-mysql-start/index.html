出现的问题

  1. 输入sudo service mysql restart出现su: warning: cannot change directory to /nonexistent: No such file or directory的解决方案

    1
2
3

    sudo service mysql stop
sudo usermod -d /var/lib/mysql/ mysql
sudo service mysql start

  2. 使用 mysql -u username -p 出现 Access denied for user 'username'@'localhost'

    进入 mysql 数据库 mysql> use mysql;
    创建新用户，使用密码验证：mysql> CREATE USER 'guest'@'localhost' IDENTIFIED BY 'guest123';
    给予新用户权限：mysql> GRANT ALL PRIVILEGES ON * . * TO 'guest'@'localhost';
    退出 mysql mysql> exit
    使用新建的用户登录 $ mysql -u guest -p
